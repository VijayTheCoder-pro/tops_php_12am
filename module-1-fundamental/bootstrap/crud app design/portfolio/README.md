## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Local server (XAMPP, WAMP, or Live Server extension)

### Installation

1. Clone or download the portfolio folder
2. Open `index.html` in your web browser
3. Or use a local server for better performance

### Using with XAMPP

1. Copy the `portfolio` folder to `xampp/htdocs/`
2. Start Apache server in XAMPP
3. Navigate to `http://localhost/portfolio/`
